In this module, we will walk you through the software you need, and how to install it. ## Command Prompt The command prompt allows you to run various commands on any machine you use. Some operating systems have a build-in command prompt, and some require a bit of extra software to get working.
Linux
Mac
Windows
Most linux distributions come with built-in command prompts. To access the command prompt, press Ctrl + Alt + T, or search for the "Terminal" application.
MacOS devices, similar to linux, come with a terminal application called "Terminal". To open your command-prompt, simply open the application.

Most of the modules in this tutorial are compatible with the default Windows shell, Powershell, which is available as an application on all Windows devices. However, down the line, this may prove to be more difficult, as most of our development and deployment is on Linux computers.

WSL2

Microsoft acknowledges that for developers, having access to Linux is an invaluable tool. WSL2 allows Windows users to run certain programs on linux, within Windows and without much hassle.

To install, open PowerShell, type the command wsl --install and click Enter.

Once WSL2 is done installing, use the Start menu to search for "Ubuntu" and open it. You should see a terminal-like application open up. From there, follow the instructions here to complete setup [1].
